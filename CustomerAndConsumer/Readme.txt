1. Compile the program using 'gcc -std=c99 -o BoundedBuffer -pthread A5.c'

2. Run the corresponding compiled fle using './BoundedBuffer x y z'
    x is the sleep length of your choice
    y is the number of producers
    z is the number of consumers

    eg. './BoundedBuffer 10 2 3'

3. After running, the result will be printed out on stdout
