This program sorts the number in the 'IntegerList.txt' file with multithread method.

1. To compile this program, use 'gcc -std=c99 -o A3 A3.c -lpthread' command. (Using c99 and lpthread in the code)

2. To run the program, use './A3' command by default, or any other compiled file name you used.

4. After running the programm, a result file named 'SortedIntegerList.txt' will be generated.

5. Open the result with 'emacs SortedIntegerList.txt' or any other editor to browse the result.

6. Generated result will be a sorted number list.
